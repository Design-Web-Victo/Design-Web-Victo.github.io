    Exercice 21 - Rainbow mode

    Pour cet exercice nous allons créer une barre de navigation "responsive" et donner l'option de changer son thème. Le thème choisie sera enregistrer si l'on veut dans le stockage local ou de session.

    Création de la barre de navigation

    Lisez l'article suivant et suivez les instructions pour arriver au même résultat. Essayez de comprendre ce qui y est fait et de ne pas seulement copier coller le code.

    • Lien vers l'article : https://dev.to/devggaurav/let-s-build-a-responsive-navbar-and-hamburger-menu-using-html-css-and-javascript-4gci

    Si l'article n'est plus disponible, j'ai recopié le code à la fin du présent document.

    Modification au menu et à la page

    Apportez les modifications suivantes au projet en cours :

    • Changez le titre WebDev. pour Rainbow mode
    • Ajoutez dans la page deux cases à cocher avec les titres suivants:
    • Enregistrer dans le stockage local
    • Enregistrer dans le stockage de session
    • Ajoutez à la fin des liens du menu une icône qui nous permettra de basculer du theme par défaut au Rainbow Theme. Voir les explication plus bas.

    Icone de basculement de theme

    Pour l'icône, vous devez utiliser le CDN de font awesome : https://fontawesome.com/. Vous n'avez qu'à vous inscrire, cliquez sur Start for free et suivre les instructions. Si tout ce passe bien vous allez avoir qu'un lien à ajouter dans votre fichier html sous la forme d'une balise script.

    • L'icône "fas fa-toggle-off" sera utilisé quand on affiche le thème par défaut
    • L'icône "fas fa-toggle-on" sera utilisé pour le rainbow theme

    Quand on clique sur l'icône, on va passer d'une à l'autre. On va aussi ajuster le thème et enregistrer le thème en cours dans le stockage selon le choix des cases à cocher.

    Modification du thème

    Pour la modification du thème, je vous laisse la liberté d'explorer la façon dont vous voulez le faire. Voici un article très complet sur le sujet : https://css-tricks.com/a-complete-guide-to-dark-mode-on-the-web/. Je vous propose ici d'utiliser la méthode des deux fichiers css telle que décrite dans l'article.

    En résumé, créez un fichier css de base qui va contenir ce qui ne change pas d'un thème à l'autre. Ensuite vous allez créer un fichier css par thème et y placer ce qui est propre à chacun (couleur du texte, du background, etc.)

    Ensuite dans votre fichier html, liez le fichier css du thème de "base" avec la balise link comme on le fait habituellement, mais en y ajoutant un id. 

    <link rel="stylesheet" href="theme_defaut.css" id="theme_css">

    Quand on va cliquer sur notre icône, on va intercepter l'événement en javascript et aller la valeur de l'attribut href pour le bon fichier css. 

    const theme = document.querySelector("#theme_css");
theme.href = "theme_rainbow.css";

    Enregistrement du thème dans le stockage

    À chaque fois qu'on clique sur l'icône de basculement de thème, on va sauvegarder lequel est choisi dans le stockage selon les options sélectionnés des cases à cocher. Si une case n'est pas cochée, effacez la valeur dans le stockage correspondant.

    Quand on va cliquer sur une des cases à cocher, on va aussi enregistrer si elle est coché ou non dans le stockage local.

    Au chargement de la page, vérifiez s'il y a une valeur dans le stockage locale, si oui chargez ce thème et ajuste l'icône ne conséquence. S'il n'y a pas de valeur, vérifiez le stockage de session et faite le même traitement. Finalement, si aucun des deux stockage ne contient de valeur, chargez le thème par défaut. Vérifiez aussi dans le stockage local si les cases doivent être coché ou non.

    Créez le thème rainbow

    C'est maintenant le temps d'être créatif, créez un thème à votre goût avec plein de couleurs et d'effets d'animation! (Vous pouvez tout aussi bien faire une version "dark mode" de la barre de navigation)

    Javascript

    const hamburger = document.querySelector(".hamburger");
const navMenu = document.querySelector(".nav-menu");
const navLink = document.querySelectorAll(".nav-link");

hamburger.addEventListener("click", mobileMenu);
navLink.forEach(n => n.addEventListener("click", closeMenu));

function mobileMenu() {
    hamburger.classList.toggle("active");
    navMenu.classList.toggle("active");
}

function closeMenu() {
    hamburger.classList.remove("active");
    navMenu.classList.remove("active");
}
