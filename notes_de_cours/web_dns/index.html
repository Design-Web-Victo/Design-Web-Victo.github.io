    Réserver un nom de domaine

    Le nom de domaine fait partie de l'adresse de votre site Web. Décomposons par exemple l'adresse suivante : https://www.cegepvicto.ca/. Le nom de domaine est cegepvicto. Ensuite on a l'extension .ca qui généralement représente le pays d'origine du site. Il existe en fait plusieurs autres extensions comme .com, .org, .net et qui peuvent être utiliser par tout le monde. Le choix du nom de domaine et de l'extension va se faire lors de la réservationi du nom de domaine. On va réserver un nom de domaine et une extension pour une année. Pendant cette année le nom de domaine nous appartient et personne ne pourra le réserver et l'utiliser. À la fin de cette période on devra renouveller le nom de domaine si on veut le conserver.

    Enregistrer un nom de domaine

    La pluppart du temps un nom de domaine vous est fournis avec votre forfait d'hébergement Web. Si ce n'est pas le cas, vous devrez utiliser les services d'un registraire de nom de domaine. Il existe plusieurs site qui offre le service en voici trois :

    • https://www.ionos.ca/
    • https://www.godaddy.com/fr-ca
    • https://www.namecheap.com/

    Sur ces sites vous avez l'option d'entrer le nom de domaine qui vous intéresse. Vous aurez ensuite une liste de domaine et d'extension disponible selon votre recherhe. Les prix varient grandement selon l'extension, à vous de voir. Une fois votre nom trouvé, il ne reste plus qu'à le réserver.
